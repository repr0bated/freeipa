# FreeIPA Gemini Context

## Focus Area
**Identity Management**

## Why Index?
Reference for replacing LDAP/Active Directory with `org.opdbus.directory`.

## Project Description
Integrated Identity and Authentication solution for Linux/UNIX networked environments.

## Key Concepts
- **LDAP**: Directory Server (389 DS).
- **Kerberos**: Authentication (MIT Kerberos).
- **PKI**: Certificate management (Dogtag).
- **DNS**: Integrated DNS management.

## Conventions
- **Language**: Python, C, JavaScript (UI).
- **Architecture**: Integrated suite of standard protocols.
